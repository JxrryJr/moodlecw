# Moodle — Coursework Template

This repository contains the starting template code for the virtual learning environment (referral/deferral) coursework.

---

## Onboarding Instructions

### Step 1: Fork the Repository
Click the **Fork** button in the top right corner of this page to create your public copy of the repository.

![Step 1: Forking the Repository](images/step1_fork.png)

### Step 2: Clone and Open in VS Code
Clone your fork to your local machine, open it in VS Code, and open the integrated terminal.

![Step 2: Opening in VS Code](images/step2_vscode.png)

### Step 3: Complete First Setup Task
- Open the [lib/widgets/nav_drawer.dart](lib/widgets/nav_drawer.dart) file.
- Replace `"Yourname Here"` with your actual name.
- Replace `"up1234567"` with your actual UP identification number.
- Save the file.

### Step 4: Run the Application
Run the following commands in the terminal to fetch dependencies and run the app in Chrome:
```bash
flutter pub get
flutter run -d chrome
```

![Step 3: Running the Application](images/step4_running.png)

### Step 5: Open Chrome Developer Tools
Press `F12` in Chrome, and toggle the **Device Toolbar** to view the application in the responsive mobile layout.

![Step 4: Emulating mobile layout in DevTools](images/step5_devtools.png)
